1. Dalam Flutter, Navigator.push() dan Navigator.pushReplacement() sama-sama digunakan untuk navigasi antarhalaman, tetapi dengan cara berbeda. Pada aplikasi Blaugrana Shop, saya menggunakan Navigator.push() di class ItemCard untuk membuka halaman Add Product. Fungsi ini menambahkan halaman baru di atas halaman sebelumnya, sehingga pengguna masih bisa kembali ke halaman utama dengan tombol “back”. Sementara itu, Navigator.pushReplacement() digunakan pada class LeftDrawer untuk navigasi utama seperti Home dan Add Product. Fungsi ini menggantikan halaman yang sedang dibuka, sehingga pengguna tidak bisa kembali ke halaman sebelumnya. Dengan begitu, Navigator.push() dipakai untuk navigasi sementara (misalnya membuka form), sedangkan Navigator.pushReplacement() digunakan untuk navigasi antarhalaman utama agar alur aplikasi lebih bersih dan efisien.
2. Saya memanfaatkan hierarchy widget seperti Scaffold, AppBar, dan Drawer untuk menjaga struktur halaman yang konsisten di seluruh aplikasi Blaugrana Shop. Setiap halaman utama saya bungkus dengan Scaffold sebagai kerangka dasar agar tata letak tetap seragam. Di dalamnya, saya menggunakan AppBar untuk menampilkan judul halaman dan memberikan identitas yang jelas pada setiap tampilan. Selain itu, saya menambahkan Drawer sebagai navigasi samping agar pengguna bisa berpindah antarhalaman seperti Home dan Add Product dengan mudah. Dengan struktur ini, setiap halaman memiliki tampilan dan navigasi yang konsisten, sehingga pengalaman pengguna menjadi lebih terarah dan nyaman.
3. Dalam desain antarmuka, penggunaan layout widget seperti Padding, SingleChildScrollView, dan ListView membantu membuat tampilan form lebih rapi, responsif, dan mudah digunakan. Pada aplikasi Blaugrana Shop, saya menggunakan Padding untuk memberi jarak antar-elemen form seperti TextFormField dan DropdownButtonFormField, sehingga tampilan tidak terlihat padat dan lebih nyaman dibaca. SingleChildScrollView saya gunakan untuk memastikan seluruh elemen form tetap bisa diakses ketika layar terlalu kecil atau ketika keyboard muncul, karena widget ini memungkinkan halaman digulir secara vertikal. Jika form berisi banyak elemen yang dinamis, ListView juga bisa digunakan untuk menampilkan daftar input dengan efisien tanpa menyebabkan overflow. Dengan kombinasi ketiga widget ini, tampilan form pada aplikasi menjadi lebih tertata, responsif, dan ramah bagi pengguna di berbagai ukuran layar.
4. Saya menyesuaikan warna tema aplikasi Blaugrana Shop agar sesuai dengan identitas visual toko yang terinspirasi dari warna khas FC Barcelona, yaitu biru dan merah. Saya menggunakan warna biru tua (indigo) sebagai warna utama pada elemen seperti AppBar, tombol, dan DrawerHeader, supaya tampilan terlihat seragam di setiap halaman. Warna sekunder seperti putih saya gunakan untuk teks agar tetap jelas dan mudah dibaca. Dengan pengaturan warna yang konsisten ini, aplikasi terlihat lebih rapi, memiliki ciri khas tersendiri, dan tetap nyaman dilihat oleh pengguna.
